Pirâmide

Uma pirâmide é uma figura geométrica tridimensional que possui uma base poligonal e faces laterais que convergem para um ponto chamado vértice.

Tronco de Pirâmide: É uma pirâmide onde uma parte é removida criando uma segunda base menor, ambas paralelas, e o espaço entre as bases é a parte restante da pirâmide. Para calcular o volume, você precisa conhecer as áreas das duas bases e a altura do tronco.
